# dotfiles
## How to install?
* `git clone https://github.com/Zeotyn/.dotfiles.git ~/.dotfiles`
* `cd ~/.dotfiles`
* `sh install.sh`
* Answer the questions

## What is included?
* tmux install & config
* vim config & plugin installs
* gitconfig
* zsh & oh-my-zsh
* zsh-theme
Install of the software is optional.
